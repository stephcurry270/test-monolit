# Консольное приложение для подбора строительных материалов

Тестовое задание на позицию разработчика в компанию Монолит. Приложение реализует цепочку выбора региона и товара с умным алгоритмом удержания клиента в случае отказа.

## 🚀 Функционал
* Динамическая загрузка каталога товаров и цен под разные регионы (СПБ, МСК, КРД).
* Интерактивный CLI-интерфейс на базе встроенного модуля Node.js `readline/promises`.
* Логика удержания: при отказе от покупки система предлагает скидку 5% или подбирает более дешевый аналог в той же категории.
* Сохранение успешных заявок в структурированные JSON-файлы.

## 🛠 Технологический стек
* JavaScript (Node.js)
* Встроенные модули `fs/promises`, `readline/promises`, `path`
